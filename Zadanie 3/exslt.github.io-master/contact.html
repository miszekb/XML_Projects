Contact

If you want to talk about the definitions of any of the extensions described on this site, or about EXSLT as an initiative, then please join the mailing list and send your messages to list@exslt.org.

If you want to suggest new extensions or submit new implementations or use cases, then please follow the submission instructions.

If you have any general comments or questions about the site or want to help out with the administrative burden, then please send your message to managers@exslt.org.
